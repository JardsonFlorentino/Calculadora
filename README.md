# 🧮 Calculadora com JavaScript

Este é um projeto de calculadora desenvolvida com **HTML, CSS e JavaScript puro**, como parte dos meus estudos de Front-End.

## 🔧 Funcionalidades

- Operações básicas: +, -, *, /
- Porcentagem (%)
- Potência (x²)
- Raiz quadrada (√)
- Inverter sinal (+/-)
- Decimal
- Limpar e calcular (=)
